# Шаг 3: Быстрый ввод транзакции

## Описание
Внедрение возможности быстрого ввода транзакции в формате "Категория Сумма Комментарий" для резкого улучшения пользовательского опыта и частоты использования бота.

## Задачи:
- Внедрить возможность быстрого ввода транзакции в формате "Категория Сумма Комментарий" в одном сообщении (например, "Продукты 256.50 хлеб, молоко")
- Добавить команду /quick для быстрого ввода с автодополнением
- Реализовать сохранение часто используемых шаблонов транзакций
