# REF-006: Исправление инициализации MorphAnalyzer [x]

## Задача
Исправить AttributeError: morph_analyzer в классах KeywordDictionary, CategoryClassifier и KeywordClassifier. Обеспечить наличие атрибута morph_analyzer во всех случаях инициализации, включая создание через __new__ и асинхронную загрузку.

## Контекст
При использовании __new__ для создания экземпляров классов, атрибут morph_analyzer может не быть инициализирован должным образом, что приводит к AttributeError при попытке его использования.

## Требования
1. Обеспечить инициализацию morph_analyzer в конструкторе __init__ и при использовании __new__
2. Обработать случаи, когда pymorphy3 не установлен
3. Обеспечить совместимость с асинхронной загрузкой данных
4. Сохранить обратную совместимость с существующим кодом

## План исправления
1. Добавить метод инициализации morph_analyzer, который можно вызывать отдельно
2. Обновить конструкторы классов для корректной инициализации morph_analyzer
3. Обеспечить восстановление состояния morph_analyzer при асинхронной загрузке
4. Добавить проверки на наличие атрибута перед использованием

## Оценка
Сложность: Средняя
Приоритет: Высокий (блокирует стабильную работу системы категоризации)
