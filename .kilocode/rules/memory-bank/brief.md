# Project Brief: Budget Bot

**Budget Bot** — это продвинутый Telegram-бот для автоматизированного ведения личного бюджета.

## Основные цели
- Упростить процесс фиксации расходов и доходов для пользователя.
- Автоматизировать распознавание бумажных чеков через QR-коды.
- Обеспечить надежное хранение данных с использованием стратегии Local-First.
- Предоставить гибкую систему категоризации транзакций на основе машинного обучения.

## Ключевой функционал
- **Распознавание чеков:** Интеграция с API Proverkacheka.com для извлечения данных из QR-кодов (сумма, товары, продавец).
- **Local-First Архитектура:** Мгновенная запись в локальную БД SQLite и фоновая синхронизация с Google Sheets.
- **Умная категоризация:** Система на базе ML (TF-IDF) и лемматизации (pymorphy3) для автоматического определения категории.
- **Гибкий ввод:** Поддержка FSM для пошагового ручного ввода и "умного" ввода через естественный язык.
- **Синхронизация:** Надежный фоновый воркер для передачи данных в Google Sheets с обработкой лимитов (Rate Limiting).
