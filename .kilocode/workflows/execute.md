# Реализация (TDD Execution)

**ШАГ 1: ПОДГОТОВКА**
1. Прочитай `.context/CURRENT_TASK.md`.
2. Проверь ветку через `git branch`. Если это `main` — СТОП.

**ШАГ 2: ЦИКЛ TDD (Red -> Green)**
1. **RED:** Создай тест-файл в `tests/` и запусти его через `pytest`.
2. **GREEN:** Напиши реализацию кода.
3. **VERIFY:** Снова запусти тесты через `pytest`.

**ШАГ 3: ЛОГИРОВАНИЕ**
1. Допиши в конец `.context/MEMORY_LOG.md` запись: `✅ UNIT TESTS PASSED`.

**ШАГ 4: ПЕРЕДАЧА**
Реализация завершена, тесты зеленые. Передаю в QA.
