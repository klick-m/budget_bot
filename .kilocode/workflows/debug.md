# Отладка (Debugging)

**ШАГ 1: АНАЛИЗ ОШИБКИ**
1. Прочитай ошибку в `.context/MEMORY_LOG.md` или выводе терминала.
2. **ВАЖНО:** Если ошибка связана с данными, используй инструмент `sqlite` (MCP), чтобы проверить реальное состояние таблицы `transactions`.
   * Пример: `SELECT * FROM transactions ORDER BY id DESC LIMIT 5;`

**ШАГ 2: ИСПРАВЛЕНИЕ**
1. Найди проблемный файл.
2. Внеси минимальные правки через `edit_file`.
3. Запусти `python scripts/smoke_test.py` для проверки фикса.

**ШАГ 3: ВОЗВРАТ**
* **ЕСЛИ ПОЧИНИЛ:**
    1. Напиши: "Баг исправлен. Возвращаю на повторный QA."
    2. **Команда:** `/workflow qa`
