# Product Description: Budget Bot

## Назначение проекта
Budget Bot - это Telegram-бот для личного и семейного учета финансов с подходом "Local-First". Основная цель проекта - предоставить пользователям удобный инструмент для быстрого ввода финансовых транзакций "на ходу" с возможностью последующего анализа данных через Google Таблицы.

## Проблемы, которые решает проект
- Неудобство ведения бюджета в традиционных приложениях
- Необходимость вводить траты вручную
- Отсутствие возможности быстро зафиксировать расходы с телефона
- Недостаток контроля над данными в облачных решениях
- Сложности с распознаванием трат без подробного описания

## Как работает система
- Пользователи могут отправлять чеки боту для автоматического распознавания
- Бот поддерживает различные способы ввода: ручной ввод, парсинг чеков, "уменный" ввод через NLP
- Все транзакции сначала записываются в локальную SQLite базу данных (Local-First подход)
- После локального сохранения происходит фоновая синхронизация с Google Таблицами
- Система включает классификацию трат по категориям с использованием машинного обучения
- Реализована система ролей (пользователь/администратор) с возможностью установки лимитов

## Целевая аудитория
- Люди, следящие за своими расходами
- Семьи, ведущие совместный бюджет
- Пользователи, предпочитающие локальное хранение данных с синхронизацией в облако
- Те, кто хочет быстро фиксировать траты "на ходу" без сложных интерфейсов
