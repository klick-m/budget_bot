# Project Brief: Budget Bot

## Core Objective
Создание Telegram-бота для личного и семейного учета финансов с подходом "Local-First". Бот должен обеспечивать мгновенную запись транзакций в локальную БД и последующую фоновую синхронизацию с Google Sheets.

## Key Features
- **Smart Input**: Распознавание текстовых запросов (например, "100 хлеб") и чеков.
- **Multi-modal Entry**: Ручной ввод, парсинг чеков, "умный" ввод через NLP.
- **Sync System**: Асинхронная выгрузка данных в Google Таблицы для долгосрочного хранения и анализа.
- **Auth & Roles**: Разграничение прав доступа (User/Admin) и установка месячных лимитов.
- **Analytics**: Генерация отчетов и визуализация трат.

## Target Audience
Пользователи, желающие быстро фиксировать расходы "на ходу" с сохранением детальной отчетности в облаке.
